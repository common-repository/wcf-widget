This rss cross feed widget plugin is for connecting the sites with each other using feeds.

== Description ==
Do you have multiple websites and want your content to redirect to each other? Now you can simply do it with a plugin without disturbing your SEO algorithm. This plugin will help you to easily show content of multiple websites on the page of your choice. You will be able to retain and deflect huge amount of traffic by single attempt. Moreover, it will help you to create a reliable brand value within multiple domains.
